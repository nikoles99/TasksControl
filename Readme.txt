Приложение для управления задачами:
1. На главной форме приложния(в Toolbar) выбрать функцию добавления новой задачи
2. На второй форме Ввести необходимые данные(Наименование, Количество часов, дату начала выполнения, дату окончания выполнения, статус задачи)
3. Нажать кнопку "save" - задача сохраниться и отобразится на главной форме в списке задач

для сборки проекта в CMD(windows):
1. Ввести путь в AndroidSdk
2. Ввести путь к gradlew
